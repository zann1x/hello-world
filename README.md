## Getting up and running

- Node.js installed
- NPM installed
- Execute *npm ci*

---

- `npm run develop` for developing on [localhost:8000](http://localhost:8000)
- `npm run deploy` for deployment on [lukaszanner.de](https://lukaszanner.de)
